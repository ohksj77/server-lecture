# server-lecture
## 멘토링 목적
- 서버의 흐름 이해
- 서버 코드와 Database 상호작용의 간략한 이해
- API 설계에 대한 이해도 높이기
- docker와 nginx의 간단한 설정법 및 사용 이유 이해

### 활용 기술
- Flask
- Swagger
- Gunicorn
- NginX
- MySQL
- docker-compose
### 유의 사항
- .env.template 주석 확인

### 실행 커맨드 (프로젝트 directory에서 CLI로 실행)
```
docker-compose up --build
```
### Swagger 링크
```
http://localhost:5001
```
### Backend API URI
```
http://localhost/api/~~커스텀URI~~
```
